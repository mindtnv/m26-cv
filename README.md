## Структура репозитория

### Лабораторные работы по компьютерному зрению

| Файл | Тема | Содержание |
|------|------|------------|
| `cv1.ipynb` | Базовые алгоритмы CV | RGB→Grayscale, свёртка, фильтры Собела, детектор Canny, HOG-дескриптор |
| `cv2.ipynb` | Локальные дескрипторы и CNN | SIFT, сопоставление особых точек, ResNet, анализ градиентов |
| `cv3.ipynb` | Детекция и сегментация | IoU, NMS, Faster R-CNN, Mask R-CNN, DeepLabv3 |
| `cv4.ipynb` | Классификация изображений | CNN на CIFAR-10, аугментация, transfer learning |

### Лабораторные работы по машинному обучению

| Файл | Тема | Содержание |
|------|------|------------|
| `f1.ipynb` | Классификация MNIST | KNN, SVM, нормализация данных |
| `f2.ipynb` | Линейная регрессия | Ridge, Lasso, ElasticNet, робастная регрессия |
| `f3.ipynb` | Классификация и регрессия | Iris, Diabetes, деревья решений, ансамбли |
